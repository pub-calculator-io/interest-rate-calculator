Free online calculator to estimate the interest rate and total interest cost of an amortized loan with a fixed monthly repayment amount.

[https://www.calculator.io/interest-rate-calculator/](https://www.calculator.io/interest-rate-calculator/)

== Usage ==

Add the Interest Rate Calculator shortcode to your page, post or sidebar:

`[ci_interest_rate_calculator]`

Add the following code to your template where you would like the Interest Rate Calculator to appear:

`<?php display_ci_interest_rate_calculator(); ?>`

== Installation ==

1. Upload the Interest Rate Calculator /ci_interest_rate_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Interest Rate Calculator plugin through the "Plugins" menu in WordPress.
